# Math MCP Server Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and publish a TypeScript MCP server (`math-mcp`) that exposes 8 math tools to AI agents, runnable via `npx math-mcp`.

**Architecture:** Stateless stdio MCP server using `@modelcontextprotocol/sdk`. Each of 8 tools lives in its own file under `src/tools/`. A shared `src/lib/math.ts` configures mathjs and exports algebrite. Tools return a consistent `{ result, numeric, latex, type }` shape or `{ error, hint }` on failure.

**Tech Stack:** TypeScript, Node.js, pnpm, mathjs, algebrite, @modelcontextprotocol/sdk, vitest

---
